v1.0 2009
  • Convert lexers.cfg into xml format.
  • Changes for wxWidgets-SVN (unicode related). Removed all wxT().
  • wxextension is now under SVN. exCommitDialog supports svn.
  • Compiles if wxUSE_STL is set.
  • Documentation is doxygen parseble, and lexers.cfg now colours doxygen comments.
  • Added exManagedFrame, and added freeze and thaw when maximize and set window size.
  • Added exStat with a ConfigDialog for colouring.
  • Added exStockArt offers stock items on the toolbar.
  • Added exSTCShell.
  • Added exToolBar.
  • Updated exApp. Added wxLocale and no longer depends on wxstd, just reads all catalogs available. Moved printer from exInterface to exApp. Moved config member from exFrame to exApp.
  • Updated exConfigDialog checkboxes grow again. wxAPPLY is supported by exConfigDialog. Added combobox items to exConfigDialog, so the find in files dialogs are no longer necessary. Uses wxSizerFlags.
  • Updated exMenu and exTool, no longer offering virtual methods.
  • Updated exStatistics. Templatized it and added Show method, and corrected Clear.
  • Updated exSTC. Added macro support. In exSTC when document is changed, uses event wxEVT_STC_MODIFIED in OnStyledText for displaying new modification time and call PropertiesMessage. If file is synchronized, while there was a selection, the selection is gone, whereas ADDTEXT should not change the selection.
  • Updated exTextfile. The ParseComments in exTextfile is now virtual, all db code in exTextFile is removed. OTL is used for grid filling.
  • Added plaform switches to lexers.cfg. Last line in lexers.cfg was not parsed correctly. Updated sql keywords in lexers.cfg.
  • Added wxUSE_GUI switches.
  • namespace no longer in header files.
  • Most enums now inside classes.
  • More default constructors, and less copy constructors. More methods are const aware.
  • wxTheApp->Yield only for main thread, otherwise calls wxThread::Yield.
  • Updated default lexer, so C++ comments counted correctly.
  • When setting new lexer, folding is enabled/disabled according to this new lexer.
  • The find item in list view popup menu did not have an icon.
  • When replacing text, the 'replace with' combobox uses the same text as default.
  • Replaced wxGenericColourButton into wxColourPickerWidget.
  • Subdir flag is used in config for config file.
  • Directory structure similar to wx.
  • Tooltip on statusbar is now set if mouse is over a pane.
