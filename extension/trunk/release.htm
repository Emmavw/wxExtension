  • Updated exTextfile. The ParseComments in exTextfile is now virtual, all db code in exTextFile is removed. OTL is used for grid filling.
  • Added plaform switches to lexers.cfg. Last line in lexers.cfg was not parsed correctly. Updated sql keywords in lexers.cfg.
  • Added wxUSE_GUI switches.
  • namespace no longer in header files.
  • Most enums now inside classes.
  • More default constructors, and less copy constructors. More methods are const aware.
  • wxTheApp->Yield only for main thread, otherwise calls wxThread::Yield.
  • Updated default lexer, so C++ comments counted correctly.
  • When setting new lexer, folding is enabled/disabled according to this new lexer.
  • The find item in list view popup menu did not have an icon.
  • When replacing text, the 'replace with' combobox uses the same text as default.
  • Replaced wxGenericColourButton into wxColourPickerWidget.
  • Subdir flag is used in config for config file.
  • Directory structure similar to wx.
  • Tooltip on statusbar is now set if mouse is over a pane.
